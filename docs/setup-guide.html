⚡ מצב נוכחי

האתר עובד כרגע עם נתוני דמו (Mock Data) - כלומר הכל עובד מקומית בדפדפן ללא שרת.

כדי להפוך את זה לאתר אמיתי עם בסיס נתונים, תשלום, ואחסון - עקוב אחרי השלבים הבאים:

1

הקמת חשבון Supabase (בסיס נתונים)

  1. היכנס ל-supabase.com ולחץ "Start your project"
  2. הירשם עם חשבון GitHub או אימייל
  3. צור פרויקט חדש (New Project):
    • - שם: birkat-hasadeh
    • - סיסמת DB: בחר סיסמה חזקה (שמור אותה!)
    • - Region: בחר אירופה (הכי קרוב לישראל)
  4. חכה שהפרויקט ייווצר (כ-2 דקות)
  5. לך ל-SQL Editor (בתפריט הצדדי)
  6. העתק את תוכן הקובץ supabase-schema.sql ולחץ Run
  7. מ-Settings → API, העתק את:
    • - Project URL
    • - anon public key
    • - service_role key (סודי! רק לשרת)
2

יצירת משתמש אדמין ראשון

  1. ב-Supabase, לך ל-Authentication → Users
  2. לחץ "Add User" → "Create New User"
  3. הכנס אימייל וסיסמה (זה יהיה חשבון הניהול שלך)
  4. לך ל-SQL Editor והרץ: 
    INSERT INTO admin_users (id, email, full_name_he, role)
VALUES (
  'UUID-של-המשתמש-שיצרת',
  'your@email.com',
  'שקד',
  'superadmin'
);
3

חיבור הסליקה (תשלום אונליין)

יש כמה אפשרויות לסליקה ישראלית:

🔵 Tranzila
הסליקה הפופולרית בישראל. צריך לפתוח חשבון ב-tranzila.com ולקבל Terminal ID.
🟢 PayMe (של קארדקום)
ממשק פשוט, מתאים לעסקים קטנים. payme.io
🟣 Stripe
פלטפורמה בינלאומית שתומכת גם בישראל. stripe.com
4

העלאה לאינטרנט (Deployment)

הדרך הפשוטה ביותר להעלות את האתר:

אפשרות א' - Vercel (מומלץ)
  1. צור חשבון ב-vercel.com
  2. העלה את קבצי הפרויקט ל-GitHub
  3. חבר את ה-repo ל-Vercel
  4. Vercel יעלה את האתר אוטומטית!
אפשרות ב' - Netlify
  1. צור חשבון ב-netlify.com
  2. גרור את תיקיית הפרויקט לדף ההעלאה
  3. האתר יהיה באוויר תוך שניות!
אפשרות ג' - דומיין משלך
  1. קנה דומיין (למשל birkat-hasadeh.co.il)
  2. חבר אותו ל-Vercel/Netlify דרך הגדרות ה-DNS
